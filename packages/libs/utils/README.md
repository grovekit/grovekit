# @grovekit/utils

Shared utilities library for the [Grovekit] IoT stack. Provides runtime
configuration loading, structured logging, async helpers, a batching worker,
datetime utilities, and typed JSON helpers used across all Grovekit packages.

> Part of the [`grovekit/grovekit`](https://github.com/grovekit/grovekit)
> monorepo. See the root [README](../../README.md) for setup and deployment
> instructions.
