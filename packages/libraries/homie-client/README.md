
# `@grovekit/homie-client`

A [GroveKit][grovekit] library for publishing and interacting with devices
implementing the [Homie MQTT convention][homie].

## Usage

Documentation for the [GroveKit][grovekit] stack is a work-in-progress. For the
time being, please refer to the examples in the [./src/examples] directory.
